Getting Started with HTML

Major Concepts

World Wide Web

Web is a collection of HTML Documents.It contains hyperlinks.Main type of document on web is HTML though it contains plain text,audios,videos,images and other stuffs.

Components of a Web

1.User's computer and Browser where browser is a program that run on a computer to display files from web.
2.Internet
3.Servers which is a computer that is optimised for hosting files.

How a web works?

When a person opens a webpage like www.udacity.com,user's computer sends a HTTP Request to a server. The server finds appropriate HTML document and sends it back to user's computer web browseris used to display it on user's computer.

HTML

HTML stands for HyperTextMarkupLanguage.
It contains:

  • Text content[what you see].
  • Markup[what it looks like].
  • Reference to other documents like images,links etc.

HTML Tags and Elements

HTML Documents are made of HTML elements. When coding in HTML we provide the type of each element by using HTML tags.

Types of HTML Tags

  • Anchor represented by "a" which is used to display links.
  • Bold represented by "b" which is used to turn the text bold.
  • Emphasis represented by "em" which is used to turn the text italic.
  • Break represented by "br" which is used to break from on line.
  • Paragraph represented by "p" which is used to write text in paragraphs.
And there are much more you can check it out in the documentation here.

Why computers called Stupid?

Computers are stupid because they interpret instructions literally. This makes them very unforgiving since a small mistake by a programmer can cause huge problems in a program.

Inline vs Block Elements

HTML elements are either inline or block. Block elements form an "invisible box" around the content inside of them.

FAQ

  • What happens if you forgot to close a tag?
    Ans:Closing tags will be automatically generated by major editors that we use.

  • What is a void Tag?
    Ans:A void Tag do not need any contents and therefore no closing tag is present inside a void Tag.

Creating a Structured Document

Major Concepts

Developer Tools(in browser)

All the elements in the page are either in the form of inline or block elements.Blocks elements encloses a "invisible box" with contents inside it.

Document Object Model[Tree-like Structure]

A cross platform and language independant convention for representing and interacting with objects in HTML.

In a browser, this structure shows up as a series of nested boxes. There are boxes inside of boxes inside of boxes, and so on...

Components for making webpage

  • HyperTextMarkupLanguage-Standard Markup language used to create webpages
  • CascadedStyleSheets-Used for describing the look and formatting of a document written in a markup language
  • Javascript-Used for the performance and backend of a webpage.

Boxes v/s HTML

Boxes that we formed from the DOM we can make them into separate divisions inside a HTML Document.

When you read an HTML document as text, you see a wave of changing indentations going up and down the left side of the document. The more indented an element is, the more deeply nested its corresponding "box" is.
